# 俄罗斯方块小游戏（本地网页版）

## 运行环境

- 已安装 **Python 3**（Windows 10 默认可很方便安装）

## 启动方式

1. 在资源管理器中打开本项目文件夹  
  （例如：`D:\demotest\俄罗斯方块小游戏 20251218A`）
2. 在地址栏输入 `powershell` 回车，打开 PowerShell 窗口
3. 在 PowerShell 中运行：

   ```powershell
   python server.py
   ```

4. 正常情况下会自动打开浏览器并访问  
   `http://localhost:8000/index.html`  
   如果没有自动打开，可手动在浏览器地址栏输入该地址。

## 操作说明

- **开始 / 重新开始**：按钮或先点击页面再直接按键操作
- **暂停 / 继续**：右侧按钮
- **← / →**：左右移动方块
- **↑**：旋转方块
- **↓**：加速下落（软降）
- **空格**：快速下落到底（硬降）

## 说明

- 游戏完全在本地浏览器中运行，无需外网连接
- `server.py` 只是在本机开了一个简单的 HTTP 服务，方便用浏览器访问游戏页面
